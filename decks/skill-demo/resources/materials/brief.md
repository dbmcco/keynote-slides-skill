# Keynote Slides Skill - Demo Presentation

## Overview
This deck demonstrates the keynote-slides-skill itself - a system for building Keynote-style HTML presentations with AI-generated infographics.

## Key Features to Showcase
1. **Single-file HTML decks** - Portable, no build step
2. **Multi-brand system** - Switch visual identity instantly
3. **AI infographics** - Generate diagrams with Gemini (nano-banana)
4. **Video generation** - Embed animated content with Veo
5. **Claude Code integration** - Evaluate and iterate on visuals
